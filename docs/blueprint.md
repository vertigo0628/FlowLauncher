# **App Name**: Flow Launcher

## Core Features:

- Auto App Categorization: Automatically sorts installed apps into predefined categories (e.g., Social, Media, Games, Utilities) for easy navigation. Implemented through machine learning to identify category
- Flower Menu: Implements a flower/radial menu design pattern for quick access to app categories, optimizing for one-handed use.
- Gesture Navigation: Supports swipe-up for app drawer and swipe-down for notifications/search, configurable by the user
- Adaptive Icons: Uses adaptive icons to ensure a consistent look and feel across different icon packs and device manufacturers.
- Theme Customization: Allows users to switch between light and dark themes, adjusting automatically based on system settings or a set schedule.
- Intelligent App Suggestions: AI-powered tool to recommend applications, the tool considers the current time and usage patterns. It displays in a dedicate area of the main display

## Style Guidelines:

- Primary color: Slate blue (#708090) for a calm, efficient feel.
- Background color: Light gray (#D3D3D3), a desaturated hue of the primary for a clean and unobtrusive backdrop.
- Accent color: Soft purple (#A090D0), analogous to the primary but with distinct saturation and brightness.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable interface.
- Use minimalist, monochromatic icons for app categories and system controls.
- Implement a clean and simple grid layout for the home screen, adjustable based on screen size and user preference.
- Use smooth, subtle animations for transitions and interactions to enhance the user experience without sacrificing performance.
